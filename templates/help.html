Using This App

This homework tracker helps you manage assignments across your courses. You can add, edit, mark complete, or delete assignments. All changes are automatically saved.

  • Add: Click “Add” to create a new assignment.
  • Edit: Click “Edit” to modify details.
  • Complete: Check the box to mark an item done.
  • Delete: Click the trash icon to remove an assignment.

If you encounter any issues, refer back to the README.md file
